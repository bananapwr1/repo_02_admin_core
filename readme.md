# Core Manager Bot (Bot #2)

Админский интерфейс для управления торговым ядром.

## Функции:
- Управление стратегиями через Supabase
- Просмотр AI-рассуждений
- Чат с Claude для обсуждения стратегий
- Получение скриншотов для анализа
- Управление парсером и авто-торговлей

## Команды:
- `/manager` - главное меню
- `/set_strategy` - установить стратегию
- `/ai_reasoning` - просмотр AI-логов
- `/chat` - обсуждение со стратегией
- `/stats` - статистика ядра

## Установка:
1. Клонировать репозиторий
2. `pip install -r requirements.txt`
3. Настроить `.env`
4. Запустить: `python main.py`
